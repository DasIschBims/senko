# senko discord bot

Build and start:

> 1
Rename `.env-example` to `.env` and fill in the required lines


> 2
```bash
git clone https://github.com/DasIschBims/senko.git
```


> 3
```bash
npm install
```


> 4
```bash
npm build
```


> 4.5
```bash
chmod 777 start.sh
```


> 5
```bash
./start.sh
```
