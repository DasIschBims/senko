# senko discord bot

Build and start:

> 1
Rename `.env-example` to `.env` and fill in the required lines


> 2
```bash
git clone https://github.com/DasIschBims/senko.git
```


> 3
```bash
npm install
```


> 4
```bash
npm build
```


> 4.5 (linux only)
```bash
chmod +x start.sh
```


> 5 (linux/bash only)
```bash
./start.sh
```

> 5 (windows)
```bash
node dist/index.js
```
